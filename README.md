## UIGraphics - Drawing - UserDefaults


This is a photo puzzle game.<br/>
The user will start off with picking a photo using either a camera captured photo, a photo from library or let the app randomly pick a stored photo.
The user can then adjust the photo position by moving, rotating and/or zooming within the 16 squares grid.<br/>
The user can share the completed puzzle picture at the end via email, social media etc...<br/>
The application counts the number of moves used to solve the puzzle.<br/>
The application has a scoring streaks system for continuous correct moves, and for incorrect moves the app reset the scoring streak and removes a point from the score unless score is at 0 points.<br/>
There is a peak "hint" feature to help the user solve the puzzle, whereby the puzzle photo appears on the screen for two seconds.
